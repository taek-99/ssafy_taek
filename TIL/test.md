## Pandas
- 1차원을 다루는 Series
- 2차원을 다루는 DataFrame

## Pandas의 장점
- 결측치 처리 용이
- 행/열 추가 삭제 간편
- 정렬
- Group By 제공
- 다양한 객체 변환
- Merge/Join 지원
- 파일 입출력 지원

## Pandas와 Numpy
- 판다스는 주로 2차원 전용
- Numpy는 다차원을 다루는데 특화

## 기초 통계량의 활용
- 이상치
    - 데이터를 분석할때 극단적으로 크거나 작은 값을 찾는 것이 중요

- 비교와 예측 가능
    - 두개 이상의 그룹을 비교할 때 어떤 차이가 있는지 시각적으로 확인 가능

- 적절한 분석방법 선택
    - 데이터의 분포 형태에 따라 평균, 중앙값등을 선택하여 분석 가능

## 모집단과 표본의 개념
- 모집단: 연구대상이 되는 전체 집단
- 표본: 모집단에서 일부를 선택하는것

## 표본조사의 필요성
- 인구조사: 전체 국민을 조사하는 방법

- 표본조사: 모든 사람을 조사할수는 없으니깐 일부만

## 표본 추출의 방법
- 단순 무작위 추출
- 충화추출
- 계통추출
- 군집추출
- 편의추출

## 분포
- 데이터가 어떤 값 주변에 어떻게 퍼져있는지를 나타내는 것입니다.

## 분포의 필요성
- 데이터의 패턴 파악
- 이상치 발견
- 비교와 예측 가능
- 적절한 분석방법 선택

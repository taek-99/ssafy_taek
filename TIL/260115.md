# 웹 크롤링

## 챕터
- 웹 크롤링
- Beutiful Soup의 활용
- Selenium의 활용

## 웹 크롤링이란?
- Web+Crawling
- 인터넷 상의 웹 페이지를 자동으로 탐색하고 데이터를 수집하는 기술
- 스크래핑
    - 특정 웹페이지에서 원하는 데이터를 추출하는 과정

## 웹 크롤링의 특징
- 자동화
- 대량 데이터 수집
- 구조화된 데이터 추출

## 웹 데이터 수집 방법
- 웹 페이지를 가져와서 필요한 정보만 추출하기
- WebDriver를 이용해 웹 브라우저 자동화 하기
- 제공된 OpenAPI를 이용해 실시간으로 데이터 가져오기

## WebDriver로 웹 브라우저 자동화
- 로그인이 필요한 서비스 등 URL만으로는 접속할수 없는 세션 유지가 필요한 작업이나 옵션이나 드롭다운 메뉴를 클릭해야함 데이터를 얻을 수 있는 작업에 필요함

# 데이터 분석을 위한 기초 통계

## 챕터 포인트
- 기초 통계량
- 조건부 확률 및 베이즈 정리
- 표본 추출과 편향
- 분포에 대한 이해


## 기초 통계량
- 위치를 추정하는 다양한 방법
    - 평균 - 모든 값을 더해 개수로 나눈 값
    - 절사평균 - 상 하위 극단값 일부를 제거하고 계산한 평균
    - 가중평균 - 각 값의 중요도를 반영해 계산하는 평균
    - 중간값 - 정렬된 데이터의 중앙 위치값
    - 가중 중간값 - 값의 가중치를 고려해 중앙 위치를 결정
    - 백분위수 - 데이터를 100등분했을때 특정 위치의 값


## 조건부 확률 및 베이즈 정리
- 베이즈 정리
    - 직접 구하기 어려움 P를 더 구하기 쉬운 게산할수 있게 해주는 공식

## 모집단과 표본의 개념
- 모집단: 연구 대상이 되는 전체 집단
- 표본: 모집단에서 일부를 선택하는것

- 표본 편향: 모집단을 잘못 대표하는 표본을 의미

## 임의 추출의 개념
- 모집단에서 특정한 규칙 없이 무작위로 표본을 선택하는 방법
- 표본추출의 종류
    - 복원추출
    - 비복원 추출
